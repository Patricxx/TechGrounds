Alle Pseudo-elementen

    Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen. 
    Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

    
Alle Pseudo-Elementen beschrijving

          

              
                  

                      Selector
                         
                      Voorbeeld
                         
                      Beschrijving
                  

              
              
                  

                      ::after
                         
                      p::after
                         
                      In CSS maakt :: after een pseudo-element aan dat het laatste kind is van het geselecteerde element. 
Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de inhoudseigenschap. 
Het is standaard inline. 
                       

                       

                          
                       

                       

                       ::backdrop
                          
                       p::backdrop
                          
                       Het :: backdrop CSS pseudo-element is een box ter grootte van de viewport die direct wordt weergegeven onder 
elk element dat in volledig scherm wordt gepresenteerd. Dit omvat zowel elementen die in de modus Volledig 
scherm zijn geplaatst met behulp van de API voor volledig scherm als < dialog > -elementen.
                       

                       

                          
                       

                       

                       ::before
                          
                       p::before
                          
                       In CSS maakt :: before een pseudo-element aan dat het eerste kind is van het geselecteerde element. 
Het wordt vaak gebruikt om cosmetische inhoud toe te voegen aan een element met de inhoudseigenschap. 
Het is standaard inline.
                       

                       

                          
                       

                       

                       ::cue
                          
                       p::cue
                          
                       Het :: cue CSS pseudo-element komt overeen met WebVTT-cues binnen een geselecteerd element. 
Dit kan worden gebruikt om bijschriften en andere aanwijzingen in media te stylen met VTT-tracks.
                       

                       

                          
                       

                       

                       ::cue-region
                          
                       p::cue-region
                          
                       Het :: cue-region CSS pseudo-element komt overeen met WebVTT-cues binnen een geselecteerd element. 
Dit kan worden gebruikt om bijschriften en andere aanwijzingen in media te stylen met VTT-tracks.
                       

                       

                          
                       

                       

                       ::first-letter
                          
                       p::first-letter
                          
                       Het :: first-letter CSS pseudo-element past stijlen toe op de eerste letter van de eerste regel van een 
blokniveau-element, maar alleen als deze niet wordt voorafgegaan door andere inhoud (zoals afbeeldingen 
of inline-tabellen). 
                       

                       

                          
                       

                       

                       ::first-line
                          
                       p::first-line
                          
                       Het :: first-line CSS pseudo-element past stijlen toe op de eerste regel van een block-level element. 
Merk op dat de lengte van de eerste regel van veel factoren afhangt, waaronder de breedte van het element, 
de breedte van het document en de lettergrootte van de tekst.
                       

                       

                          
                       

                       

                       ::file-selector-button
                          
                       p::file-selector-button
                          
                       Het :: file-selector-button CSS pseudo-element vertegenwoordigt de knop van een < input > van het type = "file".
                       

                       

                          
                       

                       

                       ::grammar-error
                          
                       p::grammar-error
                          
                       Het :: grammar-error CSS-pseudo-element vertegenwoordigt een tekstsegment dat de user-agent als grammaticaal 
incorrect heeft gemarkeerd.
                       

                       

                          
                       

                       

                       ::marker
                          
                       p::marker
                          
                       Het :: marker CSS pseudo-element selecteert het markeringsvak van een lijstitem, dat doorgaans een opsommingsteken 
of nummer bevat. Het werkt op elk element of pseudo-element dat is ingesteld om weer te geven: lijstitem, zoals 
de elementen < li > en < summary >.
                       

                       

                          
                       

                       

                       ::part()
                          
                       p::part()
                          
                       Het :: part CSS pseudo-element vertegenwoordigt elk element binnen een schaduwboom dat een overeenkomend 
    part-attribuut heeft.
                       

                       

                          
                       

                       

                       ::placeholder
                          
                       p::placeholder
                          
                       Het :: placeholder CSS pseudo-element vertegenwoordigt de placeholder tekst in een < input > of < textarea > element.
                       

                       

                          
                       

                       

                       ::selection
                          
                       p::selection
                          
                       Het :: selection CSS pseudo-element past stijlen toe op het deel van een document dat is gemarkeerd door de 
gebruiker (zoals klikken en slepen met de muis over tekst).
                       

                       

                          
                       

                       

                       ::slotted()
                          
                       p::slotted()
                          
                       Het :: slotted () CSS-pseudo-element vertegenwoordigt elk element dat in een sleuf in een HTML-sjabloon is geplaatst 
(zie Sjablonen en sleuven gebruiken voor meer informatie).
Dit werkt alleen bij gebruik binnen CSS geplaatst in een schaduw-DOM. Merk ook op dat deze selector geen tekstknooppunt 
selecteert dat in een sleuf is geplaatst; het richt zich alleen op feitelijke elementen.
                       

                       

                          
                       

                       

                       ::spelling-error
                          
                       p::spelling-error
                          
                       Het :: spelling-error CSS-pseudo-element vertegenwoordigt een tekstsegment dat door de user-agent is gemarkeerd als onjuist gespeld.
                       

                       

                          
                       

                       

                       ::target-text
                          
                       p::target-text
                          
                       Het :: target-text CSS pseudo-element vertegenwoordigt de tekst waarnaar is gescrold als de browser scroll-naar-tekstfragmenten ondersteunt.
Hiermee kunnen auteurs kiezen hoe ze dat gedeelte van de tekst willen markeren.
