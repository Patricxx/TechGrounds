Opdracht 3

    Wat gebeurt er als je een width en een height toevoegt aan de span tag ( < span > ) van vorige opdracht?

    Door height en width toe te passen verander je het formaat van de boxen waar de teksten in zitten.

Dit is een blockbox en neemt de hele breedte in waardoor de volgende box op de volgende regel zit.

In dit blok zit inline tekst in wat dus niet op de volgende regel komt te staan
