ID Selector

    An ID selector begins with a # rather than a dot character, but is used in the same way as a class selector. 
    However, an ID can be used only once per page, and elements can only have a single id value applied to them. 
    It can select an element that has the id set on it, and you can precede the ID with a type selector to only 
    target the element if both the element and ID match. You can see both of these uses in the following example:
