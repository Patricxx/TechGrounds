In het bovenstaande voorbeeld kunt je zien dat deze selectors worden gebruikt.


    Door li [class] te gebruiken, kunnen we elke selector matchen met een class-attribuut. 
    Dit komt overeen met alle lijstitems behalve de eerste.

    li [class = "a"] matcht een selector met een klasse van a, maar niet een selector met 
    een klasse van a met een andere door spaties gescheiden klasse als onderdeel van de waarde. 
    Het selecteert het tweede lijstitem.

    li [class ~ = "a"] komt overeen met een klasse van a maar ook een waarde die de klasse 
    van a bevat als onderdeel van een door spaties gescheiden lijst. Het selecteert de tweede 
    en derde lijstitems.


    li[class] {
        font-size: 200%;
    }
    
    li[class="a"] {
        background-color: yellow;
    }
    
    li[class~="a"] {
        color: red;
    }
