Opdracht 2

    CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? 
    Kun je ook uitleggen wat elk element uit de syntax betekent?

    
CSS- syntaxis

    Een CSS-regel bestaat uit een selector en een declaratieblok.
    De selector wijst naar het HTML-element dat u wilt opmaken.
    Het declaratieblok bevat een of meer declaraties gescheiden door puntkomma's.
    Elke declaratie bevat een CSS-eigenschapsnaam en een waarde, gescheiden door een dubbele punt.
    Meerdere CSS-declaraties worden gescheiden door puntkomma's en declaratieblokken worden omgeven door accolades.

    
Voorbeeld uitgelegd

    • P is een selector in CSS, het verwijst naar het HTML-element dat u wilt stijlen: < p >.
    • Color is een property, en red is de property value, dit deel samen maakt een declaration.
    • Text-align is een property en center is de property value, dit deel samen maakt een declaration.

    < p >  { color : red ;  text-align ; center ;  }
