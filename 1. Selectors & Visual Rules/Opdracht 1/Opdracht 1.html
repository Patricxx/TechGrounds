Opdracht 1


    Je kunt op 3 verschillende manieren CSS in je HTML inladen. 
    Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.
    
    Deze drie manieren zijn er mogelijk om CSS toe te voegen aan je HTML:
    
    1. Externe CSS
    2. Interne CSS
    3. Inline CSS

    
1. Externe CSS

    Met een externe stylesheet kun je het uiterlijk van een hele website veranderen door slechts één bestand te wijzigen!
    Elke HTML-pagina moet een verwijzing bevatten naar het externe stijlbladbestand in het link-element, in de head-sectie.

    
2. Interne CSS

    Een interne stijlpagina kan worden gebruikt als een enkele HTML-pagina een unieke stijl heeft.
    De interne stijl wordt gedefinieerd in het style-element, in de head-sectie.

    
3. Inline CSS

    Een inline stijl kan worden gebruikt om een ​​unieke stijl toe te passen op een enkel -element.
    Om inline stijlen te gebruiken, voegt je het style-attribuut toe aan het relevante -element. 
    Het style-attribuut kan elke CSS-eigenschap bevatten.
