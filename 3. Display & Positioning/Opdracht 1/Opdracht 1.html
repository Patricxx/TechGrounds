Opdracht 1

    Welke position values zijn er? Leg voor elke position values uit wat het doet. 
    Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

    De positioneigenschap specificeert het type positioneringsmethode dat voor een element wordt gebruikt.

    Er zijn vijf verschillende positiewaarden:
    • Static
    • Relative
    • Fixed
    • Absolute
    • Sticky

    
Position: static;

    HTML-elementen worden standaard statisch gepositioneerd.
    Statisch geplaatste elementen worden niet beïnvloed door de eigenschappen boven, onder, links en rechts.
    Een element met position: static;is niet op een speciale manier gepositioneerd; het wordt altijd gepositioneerd 
    volgens het normale verloop van de pagina:

    

        Dit element heeft position: static;
      


    
Position: relative;

    Een element met position: relative;is gepositioneerd ten opzichte van zijn normale positie.
    Door de eigenschappen boven, rechts, onder en links van een relatief gepositioneerd element in te stellen, 
    wordt het van zijn normale positie afgesteld. Andere inhoud wordt niet aangepast om te passen in een opening 
    die door het element is achtergelaten.

    

        Dit element heeft position: relative;
        


    
Position: fixed;

    Een element met position: fixed;wordt ten opzichte van de viewport gepositioneerd, wat betekent dat het altijd 
    op dezelfde plaats blijft staan, ook als de pagina wordt gescrolld. De eigenschappen boven, rechts, onder en 
    links worden gebruikt om het element te positioneren.
    Een vast element laat geen gat in de pagina waar het normaal gesproken zou zijn gelokaliseerd.
    Let op het vaste element in de rechterbenedenhoek van de pagina. Hier is de CSS die wordt gebruikt:

    

        Dit element heeft position: fixed;
        


    
Position: absolut;

    Een element met position: absolute;is gepositioneerd ten opzichte van de dichtstbijzijnde gepositioneerde voorouder 
    (in plaats van gepositioneerd ten opzichte van de viewport, zoals vast).
    Echter; als een absoluut gepositioneerd element geen gepositioneerde voorouders heeft, gebruikt het de hoofdtekst 
    van het document en beweegt het mee met het scrollen van de pagina.

    
This div element has position: relative;
        
This div element has position: absolute;

      


    
Positie: sticky;

    Een element met position: sticky;wordt gepositioneerd op basis van de scrollpositie van de gebruiker.
    Een plakkerig element wisselt tussen relativeen fixed, afhankelijk van de scrollpositie. 
    Het wordt relatief gepositioneerd totdat een bepaalde offset-positie wordt bereikt in de viewport - 
    dan blijft het op zijn plaats "plakken" (zoals positie: vast).

    
Ik ben sticky!
